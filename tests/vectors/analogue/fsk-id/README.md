# Analogue FSK ID reference vector

`reference.json` is generated by `generate_reference.py`. The generator is a standalone
Python standard-library implementation of the accepted QSSTV/mySSTV wire sequence. It
does not import, execute, or read Scanline SSTV production code.

The frozen identifier is `M6VPN-1`. It exercises letters, a digit, and punctuation. The
fixture records each exact rational event, the XOR checksum, cumulative boundaries, total
duration, event count, and cumulative frame counts at every supported project rate.

Generate and verify the fixture with:

```sh
python3 tests/vectors/analogue/fsk-id/generate_reference.py \
	> tests/vectors/analogue/fsk-id/reference.json
sha256sum tests/vectors/analogue/fsk-id/generate_reference.py \
	tests/vectors/analogue/fsk-id/reference.json
```

Protocol and implementation provenance, including pinned artifact hashes, is recorded in
`docs/protocols/analogue/fsk-id.md`.
